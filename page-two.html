Installing this frontend framework on the platform of your choice is quite simple. In order to install Materialize so it can be used for your website or other web development endeavour, simply download the Materialize (for beginners, the standard version with little extra setup required) or the Sass package (which requires a compiler and allows you to choose which components you want, since it includes the source SCSS files). These packages can be downloaded from the Materialize website.

How to Install and Implement Materialize Framework

A Quick and Simple Guide

Installing the materialize framework on the platform of your choice is quick and simple. In order to install Materialize for development, first go to the materialize website here. From there, click on the "Getting Started" tab and you will be directed to two different packages called Materialize and SASS. For beginners, we suggest downloading the standard version since it does not require much setup and includes all components that Materialize has to offer. However, if you want control over which components you want to include, then we suggest downloading SASS. Though you need a complier, the SASS package allows developers control over which components they want from Materialize, saving storage space and effort.
